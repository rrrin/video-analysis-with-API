# video analysis with API

 Анализ данных о канале на Youtube, полученных с помощью API. Работа выполнена в Jupyter Notebook на языке Python с использованием библиотек pandas, seaborn, matplotlib и др.

 Основные результаты:

 - Получение необходимых данных о канале с помощью API.

- Выделениие наиболее и наименее популярных видео с канала.

- Создание новых признаков в датафрейме

- Выделение изменения расписания выхода видео в течение всего развития канала.

- Анализ изменения популярности видео в течение всего развития канала по количеству просмотров, лайков и комментариев.
